This interface allows you to predict the likelihood of heart disease based on clinical parameters.

About This Project

This application is developed using Flask and implements a machine learning pipeline to predict heart disease based on clinical parameters. The model has been trained on a dataset containing various health metrics and has been optimized for accuracy.

The prediction is based on factors such as age, sex, chest pain type, resting blood pressure, cholesterol levels, and other clinical parameters.
